# NewsApp

## To run

Log in to [newsapi](https://newsapi.org/), copy key and paste it to `NewsApp/Common/Constants/Keys.swift` replacing `"API_KEY"`

To test offline change `api` provider to `.mock` in `NewsApp/Common/AppEnvironment/AppEnvironment.swift` in `AppEnvironment.default` declaration
